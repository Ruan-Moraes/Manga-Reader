# Manga Reader — Web UI kit

Interactive recreation of the Manga Reader SPA (React + Vite + Tailwind v4 + React Query + React Router).

## Screens
- **Home** — hero, trending carousel, community CTA, recent chapters
- **Title detail** — poster, synopsis, rating, chapter list, reviews, groups
- **Library** — tabs (Lendo, Concluídos, Em espera, Largados, Planejados)
- **Profile** — avatar header, stats, currently reading, last reviews
- **Reader** — chapter navigation + page gutter

## Components
`NavBar`, `SideMenu`, `Footer`, `MangaCard`, `Poster`, `Button`, `Badge`, `Stars`, `Icon`.

Interactive: click the menu, browse titles, add to library, navigate between screens. All state is local/in-memory.
